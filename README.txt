Программа учета животных в питомнике.

При запуске пользователь, следуя пунктам меню, может:
 -добавлять новых животных;
 -смотреть список всех;
 -добавлять команды, которым животные обучены;
 -просматривать прогресс бучения командам;

Добавлен класс Counter, показывающий кол-во всех имеющихся животных при выводе всего списка.

Для внесенной информации (дата, вид животного) предусмотрены проверки и обработка вероятных ошибок.
